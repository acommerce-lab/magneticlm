# المحوّل المثلثي الجيني — المرجع النظري

## 1. المقدمة

MagneticLM v7 هو نموذج لغوي يحاكي بنية المحولات (Transformers) باستخدام **الإحصاء فقط** — بدون انحدار تدريجي، بدون أوزان مُتعلَّمة. كل مصفوفة أوزان وكل معامل بنيوي (بُعد التضمين `d`، عدد الطبقات `L`، إسقاطات الانتباه `Wq/Wk`) يُشتق حتمياً من البيانات عبر سلسلة عمليات مبرهنة رياضياً.

**الأطروحة المركزية**: المحولات تعيد توزيع المعرفة الإحصائية المسطحة على فضاء طيفي متعدد الأبعاد. إذا أمكن حساب هذا التوزيع تحليلياً (عبر SVD) بدلاً من تكرارياً (عبر backprop)، نحصل على نموذج:
- قابل للتفسير بالكامل (كل معامل يعود لإحصائية في المتن)
- حتمي (نفس البيانات → نفس النموذج دائماً)
- مؤسس نظرياً (كل خطوة مدعومة بمبرهنة)

## 2. الأسس الرياضية

### 2.1 قياس المعرفة (شانون، 1948)

قبل بناء أي نموذج، نقيس **المعرفة الجوهرية** في البيانات من عدّ البيغرام الخام:

```
H(next|prev) = -Σ_w P(w) Σ_{w'} P(w'|w) log₂ P(w'|w)
```

هذه هي **الإنتروبيا الشرطية** — كم بت نحتاج لتخمين الكلمة التالية بمعرفة السابقة.

- **معامل المعرفة**: `K = 1 - H(next|prev) / log₂(V)` حيث `V` = حجم القاموس
- **الحد الأدنى لـ PPL**: `PPL_bound = 2^H` — لا نموذج يتجاوز هذا الحد (مبرهنة ترميز المصدر لشانون)
- **مبرهنة الغطاء**: الاستخراج المستقر يتطلب `N > V · ln(V)` توكن

مثال: WikiText-103 بـ 860K سطر أعطى `K = 53.3%`، `PPL_bound = 157.2`، تغطية = 191x.

### 2.2 PPMI — المعلومات المتبادلة النقطية الموجبة

**التعريف**: `PPMI(a,b) = max(0, log(P(a,b) / P(a)·P(b)))`

**المبرر (شانون)**: PPMI هو **المقياس الوحيد** الذي يحقق:
1. صفر عند الاستقلال
2. متماثل: `I(a;b) = I(b;a)`
3. لا يتأثر بتكرارات الكلمات المنفردة

لا يوجد مقياس آخر يحقق الثلاثة معاً. هذا مُبرهن في نظرية المعلومات.

**رابط Yoneda**: في جبر الفئات، مبرهنة Yoneda تقول: "أي كائن يُحدَّد تماماً بعلاقاته مع كل الكائنات الأخرى." صف PPMI للكلمة `w` هو بالضبط هذا: هوية الكلمة هي مجمل علاقاتها. التضمين **وفيّ** (injective): كلمات مختلفة لها ملفات مختلفة.

### 2.3 SVD — تحليل القيمة المفردة

**مبرهنة Eckart-Young (1936)**: لأي مصفوفة M وأي رتبة مستهدفة d، SVD المبتور يعطي أقل خطأ إعادة بناء بمعيار Frobenius. **لا يوجد تقريب أفضل.** هذه مبرهنة مُثبتة.

تطبيقاً على PPMI: `PPMI ≈ U × diag(S) × V.T`
- كل قيمة مفردة `S_i` تمثل **دائرة معرفة** — نمط مستقل في البيانات
- `S_1` (الأكبر): النمط المهيمن (غالباً البنية النحوية)
- `S_d` (الأصغر المحتفظ به): أضعف نمط مُبقى
- التضمينات: `E = U[:,:d] × diag(√S[:d])` — كل كلمة نقطة في فضاء طيفي d-بُعدي

### 2.4 اختيار البُعد

**حد ضوضاء Marchenko-Pastur**: لمصفوفة عشوائية V×V بـ E مدخل غير صفري وتباين σ²:
```
S_noise = σ_ppmi × √(2E/V)
```

**المشكلة**: أطياف PPMI تتناقص تدريجياً — حد الضوضاء يعطي `d ≈ V` (ضخم جداً). يفصل الإشارة عن الضوضاء لكن ليس الإشارة **المفيدة** عن **المُضرّة**.

**الحل: عتبة التباين التراكمي**:
```
cumsum(S²) / sum(S²) ≥ var_target → d
```
مع `var_target = 0.3` (احتفظ بـ 30% من الطاقة الطيفية):
- 100 سطر → d=21، 1K → d=104، 860K → d=186

### 2.5 التحليل الطيفي لمعامل الانتقال

مصفوفة انتقال البيغرام `T[i,j] = P(w_j | w_i)` تلتقط الأنماط التسلسلية. نسقطها في فضاء التضمين:

```
M_embed = E.T × T × E    [d × d]
```

SVD لـ M_embed: `U_m × diag(S_m) × V_m.T`
- `U_m`: دوران طيفي للاستعلامات (ماذا تتوقع الكلمة)
- `V_m`: دوران طيفي للمفاتيح (ماذا تمثل الكلمة)
- `S_m`: **قوى الانتقال** — مدى قوة ربط كل دائرة طيفية للكلمات تسلسلياً

**بناء Wq و Wk**:
```
Wq = U_m × diag(√S_dampened)
Wk = V_m × diag(√S_dampened)
```

**النتيجة الحاسمة**: `Q × K.T = x.T × M_embed_dampened × x` — درجات الانتباه تقيس **التشابه المرجّح بالانتقال**، وليس مجرد تشابه الكوسين.

### 2.6 مخروط التقطير الطيفي

كل طبقة تطبّق تخميداً أقوى تصاعدياً:
```
α_l = 2^l    (هندسي: 1, 2, 4, 8, ...)
```

- الطبقة 0 (α=1): كل الدوائر تساهم (استكشاف)
- الطبقة L-1 (α=2^{L-1}): فقط الدوائر القوية (تقطير)

**عدد الطبقات**: `L = ceil(log₂(H_max / H))` — مشتق من نسبة الإنتروبيا. كل طبقة تحذف نصف الإنتروبيا المتبقية (r=2 مبرر بنظرية Rate-Distortion).

### 2.7 البنية المثلثية

الأبعاد تتقلص في كل طبقة: `d_l = d // 2^l`

- البتر عبر القطع صحيح لأن أبعاد SVD مرتبة بالأهمية
- تمثيل الكلمة عبر الطبقات = **مثلث** (عرض متناقص) لا مستطيل
- أكفأ حسابياً وأصدق تمثيلياً

## 3. البنية المعمارية

### 3.1 خط الأنابيب
```
بيانات → قاموس → ترميز → قياس المعرفة(H, K, PPL_bound, تغطية)
  → PPMI → SVD(d من var_target) → طيف(Wq, Wk) → محوّل مثلثي → تقييم
```

### 3.2 الانتباه
```
Q = x × Wq,  K = x × Wk,  V = x
scores = Q × K.T / √d  (مع قناع سببي)
output = softmax(scores) × V
x = LayerNorm(x + output)
```

### 3.3 الشبكة الأمامية الإحصائية
```
h = ReLU(x × E_norm.T / √d)    → تنشيط الكلمات
h = h / sum(h)                  → تطبيع
ffn = h × E                     → مجموع مرجّح للتضمينات
x = LayerNorm(x + ffn)
```

### 3.4 رأس الإخراج
```
logits = x[-1] × E.T    (ضرب خام — نمط GPT-2)
```

## 4. ما أُثبت وما لم يُثبت

### 4.1 مبرهنات صلبة
| الادعاء | المبرهنة | الحالة |
|---------|---------|--------|
| PPMI مقياس أمثل | وحدانية MI (شانون) | **مُثبت** |
| SVD ضغط أمثل | Eckart-Young (1936) | **مُثبت** |
| PPL_bound حد حقيقي | مبرهنة ترميز المصدر | **مُثبت** |
| N > V·ln(V) للاستقرار | مبرهنة الغطاء | **مُثبت** |

### 4.2 نتائج تجريبية
| النموذج | d | PPL (860K) | kingdom |
|---------|---|-----------|---------|
| v6 كوسين بسيط | 100 | 42,345 | **المرتبة 1** |
| v7 d=4096 | 4096 | 364 مليار | المرتبة 1 |
| v7 d=186 | 186 | 322 مليون | المرتبة 27 |
| v7 d=8 | 8 | 96,080 | >50 |
| الحد النظري | — | 157 | — |

**النتيجة الأهم**: v6 البسيط يتفوق على v7 المعقد. السبب: الأوزان الإحصائية غير مُحسَّنة للتنبؤ.

### 4.3 مسائل مفتوحة
- **var_target**: لا صيغة مغلقة للقيمة المثلى
- **ميل المخروط**: r=2 مبرر لكن ليس مُثبتاً كأمثل وحيد
- **الفجوة الجوهرية**: أوزان إحصائية ≠ أوزان مُتعلَّمة

## 5. المقارنة مع المحولات المعيارية

| المكوّن | المحول المعياري | MagneticLM v7 |
|---------|----------------|---------------|
| التضمينات | مُتعلَّمة (backprop) | SVD على PPMI |
| d | معامل يدوي | تلقائي (تباين تراكمي) |
| عدد الرؤوس | معامل يدوي | d (كل بُعد = رأس) |
| عدد الطبقات | معامل يدوي | تلقائي: ceil(log₂(H_max/H)) |
| Wq, Wk | مُتعلَّمة | من SVD لـ E.T×T×E |
| رياضيات الانتباه | softmax(QK.T/√d)V | **مطابقة** |
| الإخراج | h × E.T | **مطابق** |
| التدريب | ساعات/أيام | ثوانٍ (تمرير واحد) |

## 6. تطبيقات على محولات قائمة

### 6.1 فهم أوزان المحول
استخراج Wq و Wk من محول مُدرَّب، حساب SVD(Wq × Wk.T)، مقارنة مع SVD(E.T × T × E) الإحصائي. التطابق يقيس: كم أعاد المحول اكتشاف الإحصاء مقابل ما تعلّمه جديداً.

### 6.2 تهيئة الأوزان
تهيئة Wq من SVD لـ E.T × T × E قبل التدريب — "بداية دافئة" تختصر التدريب.

### 6.3 ضغط النماذج
إذا تطابقت الدوائر الطيفية المُتعلَّمة مع الإحصائية، يُقلَّم الباقي.

### 6.4 تصميم معماري
الفجوة الطيفية للبيانات تقترح d وعدد الرؤوس وL.

## 7. التداعيات النظرية

- **المحولات = توزيع المعرفة المسطحة على فضاء متعدد الأبعاد**
- **منظور Yoneda**: الكلمة هي علاقاتها
- **المعرفة = انخفاض العشوائية**: K = 1 - H/H_max
- **التمثيل المثلثي**: المعنى له تراتب دقة (من العام للخاص عبر الطبقات)

## 8. القيود

1. أوزان إحصائية ≠ أوزان مُتعلَّمة — الفجوة الجوهرية
2. عنق زجاجة البيغرام: كل المعرفة من P(next|prev)
3. var_target لا يزال معاملاً حراً
4. تكلفة SVD على V×V عالية
5. لا دلالة تركيبية: "ليس حاراً" ≠ "حار"

## 9. اتجاهات مستقبلية

1. انتقالات أعلى رتبة (ثلاثيات، رباعيات)
2. تنقيح تكراري (تمريرات متعددة كعصور تدريب)
3. خروج مبكر بناءً على الثقة
4. دوائر طيفية هرمية (دوائر الدوائر)
5. تشكيل فئوي كامل (تضمين Yoneda، دلالة فنكتورية)
6. تهيئة هجينة: أوزان إحصائية → تدريب تدريجي
